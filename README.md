# ShiftDox - Document Converter

ShiftDox is a fast and secure document conversion application. It features a beautiful, responsive frontend built with React (Vite) and a robust backend built with Ruby on Rails (API mode).

## Features
- **Premium UI:** Glassmorphism design with smooth micro-animations.
- **Authentication:** Login, Register, or Continue as Guest.
- **Drag & Drop:** Easy file uploading for document conversions.
- **Format Support:** PDF, Word, TXT, and Image conversions (implementation in progress).

## Tech Stack
- **Frontend:** React, TypeScript, Vite, React Router, Lucide Icons, Vanilla CSS.
- **Backend:** Ruby on Rails (API Mode).

## Getting Started

### Prerequisites
- Node.js
- Ruby (3.3+)
- Rails

### Running the Frontend
```bash
cd frontend
npm install
npm run dev
```

### Running the Backend
```bash
cd backend
bundle install
rails server
```
